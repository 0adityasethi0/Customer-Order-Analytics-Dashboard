📊 Customer Order Analytics Dashboard

An interactive business analytics dashboard built using Streamlit, Pandas, and Plotly to analyze customer orders, product performance, and revenue trends.

The dashboard allows users to upload their own Excel data and generate real-time insights with interactive filters and visualizations.

**FEATURES**

Core Analytics:

Customer Summary (First Order, Last Order, Total Sales)
Days Since Last Order tracking
Repeat vs New Customer analysis

Product Insights:

Top Selling Products
Revenue by Product

Business Insights:

Revenue by Location (Delhi, Gurgaon, Noida, etc.)
Payment Mode Breakdown
Delivery Status Tracking

Time Analysis:

Monthly Sales Trend

Filters:

Search by Customer Name
Search by Phone Number
Date Range Filtering

Export:

Download filtered dataset as CSV

Data Input:

Upload Excel file directly (no database required)


**Tech Stack Used:**
Python
Streamlit
Pandas
Plotly
