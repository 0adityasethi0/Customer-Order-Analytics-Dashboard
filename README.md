📊 Customer Order Analytics Dashboard
An interactive business analytics dashboard built using Streamlit, Pandas, and Plotly to analyze customer orders, product performance, and revenue trends.
The dashboard allows users to upload their own Excel data and generate real-time insights with interactive filters and visualizations.

                                        FEATURES

**Core Analytics:**
1. Customer Summary (First Order, Last Order, Total Sales)
2. Days Since Last Order tracking
3. Repeat vs New Customer analysis

**Product Insights:**
1. Top Selling Products
2. Revenue by Product

**Business Insights:**
1. Revenue by Location (Delhi, Gurgaon, Noida, etc.)
2. Payment Mode Breakdown
3. Delivery Status Tracking

**Time Analysis:**
Monthly Sales Trend

**Filters:**
1. Search by Customer Name
2. Search by Phone Number
3. Date Range Filtering

**Export:**
Download filtered dataset as CSV

**Data Input:**
Upload Excel file directly (no database required)


                                        TECH STACK USED
Python
Streamlit
Pandas
Plotly
